# End-to-End Encryption (XOR Logic)

This project demonstrates a simple End-to-End Encryption simulation using XOR logic in Python.

## How It Works
1. User 1 sends a message
2. A random secret key is generated automatically
3. Message is encrypted using XOR logic
4. User 2 decrypts the message using the same key

## Features
- Automatic secret key generation
- XOR-based encryption and decryption
- Simple 3-phase communication model

## Technologies Used
- Python
- Random module
- Bitwise XOR operator (^)
